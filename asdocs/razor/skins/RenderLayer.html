drawRect()method
protected function drawRect(x:Number, y:Number, w:Number, h:Number, r:* = 0, c:* = -1, alpha:* = 1, rot:* = 0, gradient:String = linear, ratios:Array = null, border:Number = -1, thickness:Number = 1):void

Draw a stylized rounded rectangle.

Parameters

x:Number — The X position of the top-left corner
 
y:Number — The Y position of the top-left corner
 
w:Number — The width of the rectangle
 
h:Number — The height of the rectangle
 
r:* (default = 0) — The radius of the rounded corners, can be a number or object of the form {tl: X, tr: X, bl: X, br: X}
 
c:* (default = -1) — The fill color, can be a number or an array of colors in order to draw a gradient
 
alpha:* (default = 1) — The fill alpha, can be a number or an array of alphas corresponding to the array of colors.
 
rot:* (default = 0) — The rotation of the gradient in degrees, or the gradient matrix object:flash.geom.Matrix;
 
gradient:String (default = linear) — The type of gradient, "radial" or "linear".
 
ratios:Array (default = null) — The ratio array of the gradient.
 
border:Number (default = -1) — The color of the border, (use -1 or undefined for no border)
 
thickness:Number (default = 1) — The thickness of the border
