Base class for a StyleSheet definition. Each StyleSheet represents a rule for a component element. You can nest StyleSheets to create a chain of selectors ([Button -> Label] for example). You can extend this class with your own style rules, or create rules dynamically by passing properties into the constructor. StyleSheet subclasses can have 3 special static properties:

baseClass:Class = the class that will be instantiated when a control is created from this particular rule

style:Object = Styles that will be applied from this particular rule.

states:Object = An object containing StyleSheets for different states (Up, Over, Down, etc.), referenced by name.
